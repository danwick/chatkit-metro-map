# ChatKit Metro Map Demo

An AI-powered metro map planner demo built with OpenAI ChatKit.

## Features

- React Flow visualization of metro routes
- Interactive route planning via chat
- Real-time map updates
- Station and line management
- Streaming AI responses

## Running on Replit

1. Import this repo on Replit
2. Add your `OPENAI_API_KEY` in Secrets
3. Click Run

## Local Development

```bash
# Set your API key
export OPENAI_API_KEY="your-key"

# Run the app
bash run.sh
```
